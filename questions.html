What kinds of questions can we ask in German?

There are two types of questions we can ask in German (and in most languages). We can either ask yes/no questions or we can ask specific questions using question words.

What are the question words that we can use in German?

We can use

  • Wann (when)
  • Wo (where)
  • Was (what)
  • Wer (wen, wem) (who)
  • Warum (wieso, welhalb) (why)
  • Wie (how)
  • Welch (which)
